> Современная онлайн-аптека, разработанная с учетом принципов инклюзивного дизайна. Доступна для людей с особыми потребностями: слабовидящих, пожилых пользователей, людей с ограниченной моторикой и когнитивными особенностями.

![Скриншот сайта](screenshot.png)

## ✨ Особенности

### Инклюзивный дизайн (WCAG 2.1 AA)
- 🔤 **4 размера шрифта** — от обычного до максимального
- 🎨 **3 темы оформления** — светлая, тёмная, высокий контраст
- 🔇 **Отключение анимаций** для пользователей с вестибулярными нарушениями
- 🔊 **Озвучивание текста** — Web Speech API читает описания товаров
- 👆 **Увеличенные кнопки** для людей с ограниченной моторикой
- ⌨️ **Полная навигация с клавиатуры**
- 🔗 **Skip-link** для скринридеров

### Функционал
- 🎤 **Голосовой поиск** — скажите название лекарства или симптом
- 🛒 **Корзина** с боковой панелью
- 📦 **Модальные окна** товаров с полным описанием
- 👨‍⚕️ **Онлайн-консультация** фармацевта
- 🔔 **Toast-уведомления**
- 📱 **Адаптивный дизайн** — mobile-first
